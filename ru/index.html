Сквозное шифрование

Вся синхронизация между устройствами использует AES-256-GCM. Данные шифруются на устройстве до передачи.

Сервер не видит данных

Наши серверы передают зашифрованные данные, но не видят содержимого. Мы не можем прочитать ваши желания и настроение.

Локальная архитектура

Всё работает на вашем устройстве. Приложение полностью функционирует офлайн, данные остаются на телефоне.

Аккаунт не обязателен

Приложение полностью работает без регистрации. Вход по желанию - для синхронизации и бэкапа. Аналитика анонимна, личные данные не передаются.
