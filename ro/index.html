Confidentialitatea este un limbaj al iubirii

Datele voastre cele mai intime merita cea mai puternica protectie.

Criptare end-to-end

Toata sincronizarea intre dispozitive foloseste AES-256-GCM. Datele sunt criptate pe dispozitiv inainte de transmitere.

Server fara acces la date

Serverele noastre transmit date criptate, dar nu vad niciodata continutul. Nu putem citi dorintele sau starile voastre.

Totul pe dispozitiv

Totul ruleaza pe dispozitivul vostru. Aplicatia functioneaza complet offline, datele raman pe telefon.

Cont optional, fara urmarire

Aplicatia functioneaza complet fara inregistrare. Autentificarea optionala deblocheaza sincronizarea si backup-ul. Analiza este anonima, datele personale nu sunt transmise.
